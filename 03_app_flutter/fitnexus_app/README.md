# FitNexus Coach Flutter App

This directory contains the Flutter application for FitNexus Coach.

## Setup

1. Install Flutter from https://flutter.dev
2. Run `flutter pub get` in this directory
3. Run `flutter run` to start the app

## Notes

- The app entrypoint is `lib/main.dart`
- Use `flutter create .` later if you want to generate platform folders or regenerate missing files
