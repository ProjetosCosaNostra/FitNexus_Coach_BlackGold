# Escopo MVP

## Profissional
- Login
- Perfil
- Cadastro de alunos
- Criação de treinos A/B/C
- Exercícios com séries, repetições, carga, descanso e observações
- Link/QR Code para aluno
- Acompanhamento básico

- Acesso pelo celular
- Visualização do treino
- Marcar exercício concluído
- Registrar carga
- Cronômetro de descanso
- Histórico simples
