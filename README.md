# FitNexus Coach BlackGold

SaaS fitness para personal trainers, professores e academias entregarem treinos digitais aos alunos por Web/PWA e futuramente Android.

## Direção do produto
- SaaS completo, não serviço manual.
- Flutter Web/PWA primeiro.
- Android/Play Store depois.
- Supabase como backend/banco.
- Landing page separada para vendas.
- Venda pela internet com tráfego pago controlado.

## Produto inicial
Plataforma para criar, entregar e acompanhar treinos digitais pelo celular.

## Público inicial
Personal trainers, professores de musculação e consultorias fitness online.
